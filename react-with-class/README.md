### Initiation

1. Start React

    ```bash
    # create react app
    npx create-react-app 'project name'
    cd project name
    npm start
    ```

### Set dependencies

1. [**axios**](https://axios-http.com/kr/docs/intro)

    > node.js와 브라우저를 위한 Promise 기반 HTTP 클라이언트 
    > 운영 환경에 따라 브라우저의 XMLHttpRequest 객체 또는 Node.js의 http api 사용

    ```bash
    # axios
    npm install axios
    ```
